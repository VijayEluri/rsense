RSense
======

*RSense* is a most useful development tools for [Ruby][].

![](rsense.png)

Features
--------

Available main features are:

* Code completion
* Type inspection

In the future, the following features will be also available:

* Static code checking
* Definition jump
* Caller jump
* Refactoring

Download
--------

There is no downloads yet.

You can access a source code from git repository.
